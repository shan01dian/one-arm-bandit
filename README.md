# 放置老虎机游戏

一个基于HTML5 Canvas的放置类老虎机游戏。

## 功能特点

- 游戏场景占据整个页面
- 老虎机每10秒自动抽奖一次，用户点击可立即抽奖
- 抽奖间隔会在每次抽奖后重新计时
- 获得的金币可用于购买更高级的老虎机
- 新老虎机具有更快的抽奖速度
- 新老虎机可以自由放置在场景中的任何位置
- 随着升级更多款式，旧款老虎机支持出售功能
- 支持大数字存储（超过21亿）

## 技术栈

- HTML5 + CSS3 + JavaScript
- Canvas 2D渲染
- LocalStorage 本地存储
- BigInt 大数字处理

## 如何运行

1. 克隆项目到本地
2. 打开 `one-arm-bandit/index.html` 文件即可开始游戏

## 游戏玩法

1. 初始时有一个基础老虎机，每10秒自动抽奖
2. 点击商店按钮购买更高级的老虎机
3. 拖动老虎机到合适的位置
4. 出售不需要的老虎机获取金币
5. 不断升级老虎机，提高金币获取速度
